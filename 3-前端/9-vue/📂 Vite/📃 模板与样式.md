## Pug
```bash
yarn add -D pug
```
示例：
```vue
<template lang="pug">
.com
  .a Hello
</template>
```

<a name="BarK0"></a>
## Less
```bash
yarn add -D less
```
示例：
```vue
<template lang="pug">
.com
  .a Hello
</template>

<style lang="less">
@color: #f00;

.com {
  .a {
    color: @color;
  }
}
</style>

```

<a name="wlgDk"></a>
## Sass/Scss
```bash
yarn add -D sass
```
示例：
```vue
<template lang="pug">
.com
  .a Hello
</template>

<style lang="scss">
$color: #f00;

.com {
  .a {
    color: $color;
  }
}
</style>
```
