# 快捷键

打开/关闭终端   Ctrl+`  

### 通用型

| 说明                   | 快捷键       |
| ---------------------- | ------------ |
| 复制                   | ctrl + c     |
| 粘贴                   | ctrl + v     |
| 删除选中内容，删除整行 | **ctrl + x** |
| 撤销-undo              | ctrl + z     |
| 保存                   | ctrl + s     |
| 全选                   | ctrl + a     |

### 编辑相关

| 说明                                                         |                              |
| ------------------------------------------------------------ | ---------------------------- |
| 将当前行的内容向上/向下复制一行                              | alt+shift+↑/↓                |
| 将当前行的内容向上/向下移动一行                              | Alt + ↑/↓                    |
| 多光标                                                       | alt+shiftt+ 按住鼠标左键拖动 |
| 重命名变量名/文件名                                          | F2                           |
| 按一次是选中光标所在的当前单词，当两次就是将下一次出现的地方也选中 | **ctrl+d**                   |

### 控制相关

| 说明     | 快捷键   |
| -------- | -------- |
| 打开终端 | Ctrl + ` |
| 控制面板 | F1       |
