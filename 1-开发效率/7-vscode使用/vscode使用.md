# 快捷键

## 通用型

| 快捷键       | 说明                   |
| ------------ | ---------------------- |
| ctrl + c     | 复制                   |
| ctrl + v     | 粘贴                   |
| **ctrl + x** | 删除选中内容，删除整行 |
| ctrl + z     | 撤销-undo              |
| ctrl + s     | 保存                   |
| ctrl + a     | 全选                   |

## 编辑相关

| 快捷键                       | 说明                                                         |
| ---------------------------- | ------------------------------------------------------------ |
| **alt+shift+↑/↓**            | 将当前行的内容向上/向下复制一行                              |
| **Alt + ↑/↓**                | 将当前行的内容向上/向下移动一行                              |
| alt+shiftt+ 按住鼠标左键拖动 | 多光标                                                       |
| F2                           | 重命名变量名/文件名                                          |
| **ctrl+d**                   | 按一次是选中光标所在的当前单词，当两次就是将下一次出现的地方也选中 |
| **ctrl + enter**             | 光标直接跳到下一行                                           |
| **ctrl + shift +  enter**    | 光标直接跳到上一行                                           |

## 常用快捷键

| 快捷键                   | 描述                     |
| ------------------------ | ------------------------ |
| **Ctrl + Shift + P /F1** | 显示命令面板             |
| **Ctrl + `**             | 打开终端                 |
| Ctrl + Shift +U          | 显示输出面板             |
| Ctrl + P                 | 按名称搜索文件，快速打开 |
| Ctrl + Shift + N         | 新建窗口实例             |
| Ctrl + Shift + W         | 关闭窗口实例             |
| Ctrl + ,                 | 用户设置                 |
| Ctrl + K Ctrl + S        | 键盘快捷方式             |
| Ctrl + B                 | 显示/隐藏侧边栏          |
| F11                      | 切换全屏                 |

## 文件操作

| 快捷键            | 描述                 |
| ----------------- | -------------------- |
| Ctrl + N          | 新建文件             |
| Ctrl + O          | 打开文件             |
| Ctrl + S          | 保存文件             |
| Ctrl + Shift + S  | 文件另存为           |
| Ctrl + W          | 关闭文件             |
| Ctrl + K Ctrl + W | 关闭所有文件         |
| Ctrl + F4         | 关闭                 |
| Ctrl + Shift + T  | 重新打开关闭的编辑器 |

| 快捷键              | 描述            |

## 导航和替换操作
