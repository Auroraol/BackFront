# 聊聊Object obj = new Object()

### JOL证明

JOL = Java Object Layout

引入的POM文件

```xml
<!--
官网: http://openjdk.java.net/projects/code-tools/jol/
定位：分析对象在JVM的大小和分布
-->
<dependency>
    <groupId>org.openjdk.jol</groupId>
	<artifactId>jol-core</artifactId>
	<version>0.9</version>
</dependency>
```
